# Will You Be My Easter Egg? 💚

A special Easter proposal website featuring beautiful images and an interactive decision prompt.

## Features
- Responsive single-page design
- Beautiful background imagery throughout
- Interactive YES/MAYBE/NO buttons
- Email integration for responses
- Semi-transparent overlays for text readability

## How to Run Locally
```bash
# Option 1: Python built-in server
python3 -m http.server 8000

# Option 2: VS Code Live Server
# Right-click index.html → "Open with Live Server"

# Option 3: Node.js
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Deployment
Hosted on Netlify. Connect your repo and deploy automatically.

## Files
- `index.html` - Main page
- `assets/css/main.css` - Styles
- `assets/js/` - JavaScript libraries
- `easter-photo-*.jpg` - Memory photos
